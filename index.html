¿Qué es el Dominio de una Función?

El dominio de una función es el conjunto de todos los valores de x para los que la función está definida y tiene valores reales.

⚠️ Restricción 1: Denominadores

En fracciones, el denominador nunca puede ser cero.

$$f(x) = \frac{1}{x - 2}$$

Si $x = 2$ → denominador = 0 ❌

Entonces: $x \neq 2$

Dominio: $\mathbb{R} - \{2\}$ o $(-\infty, 2) \cup (2, +\infty)$

√ Restricción 2: Raíces Pares

Las raíces pares ($\sqrt{\phantom{x}}$, $\sqrt[4]{\phantom{x}}$, ...) solo existen para valores ≥ 0.

$$f(x) = \sqrt{x - 3}$$

Necesitamos: $x - 3 \geq 0$

Entonces: $x \geq 3$

Dominio: $[3, +\infty)$

ln Restricción 3: Logaritmos

El logaritmo solo existe para valores > 0 (estrictamente positivos).

$$f(x) = \ln(x - 1)$$

Necesitamos: $x - 1 > 0$

Entonces: $x > 1$

Dominio: $(1, +\infty)$
